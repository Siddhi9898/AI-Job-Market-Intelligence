# AI Job Market Intelligence Platform

This project analyzes AI and Data Science job market trends using a Streamlit dashboard.

## 🔗 Live Demo
https://your-app-link.streamlit.app

## Features
- Job Market Analysis
- Skill Demand Analysis
- Role Skill Analyzer
- Salary Analysis
- Salary Predictor
- Skill Gap Analyzer
- Skill Roadmap Generator
- Resume Skill Analyzer

- Job Recommender
- AI Job Trend Prediction

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
The dataset contains job postings with:
- Job Title
- Company
- Location
- Skills
- Salary information

## Run Locally

Clone the repository

git clone https://github.com/Siddhi9898/AI-Job-Market-Intelligence.git

Navigate to the project folder

cd AI-Job-Market-Intelligence

Install dependencies

pip install -r requirements.txt

Run the Streamlit app

streamlit run app.py
